## Installation

    git clone https://github.com/lighttigerXIV/Catppuccin-Ulauncher.git
    cd Catppuccin-Ulauncher
    cp -r Catppuccin Catppuccin-Round ~/.config/ulauncher/user-themes
    
Now you can select the theme and enjoy it :D
